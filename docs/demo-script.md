# Demo Script

## 0:00-0:15

0G builders need a public way to prove real integration quality, not just submit a repo and screenshots.

## 0:15-0:35

Open the app and submit a project with GitHub repo, demo, 0G contract address, Explorer link, target network, and claimed 0G modules.

## 0:35-1:05

Show the analysis job. Explain that specialized agents inspect integration proof, repo quality, docs, demo readiness, security, and community usefulness.

## 1:05-1:30

Open the BuildProof Passport. Show score, badges, risks, fixes, and judge summary.

## 1:30-1:55

Open the 0G Storage proof. Show the report root and explain that the full report is stored off-chain on 0G Storage.

## 1:55-2:20

Open the 0G Chain Explorer proof. Show the registry transaction and report hash.

## 2:20-2:40

Open Judge Mode. Show that every HackQuest proof item is visible on one page.

## 2:40-2:55

Open the leaderboard. Explain that this can become ecosystem infrastructure for hackathons, grants, and project discovery.

## 2:55-3:00

Close with: BuildProof makes 0G project quality verifiable.
